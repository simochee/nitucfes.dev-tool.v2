ドキュメント

条件分岐

  • center

    中央のフォトリフレクタ

  • left

    左側のフォトリフレクタ

  • right

    右側のフォトリフレクタ

  • black

    センサが黒を検知している状態

  • white

    センサが白を検知している状態

  • if(all)

    条件分岐がない場合に必ず記述

  • 条件演算子は==のみ使用できます

動作

  • car.fwd;

    前進させる

  • car.rev;

    後進させる

  • car.stop;

    停止させる

  • car.right;

    右旋回させる

  • car.left

    左旋回させる

  • led.flash

    中央のLEDを点滅させる

キーボードショートカット

  • ctrl + s

    コンパイル結果をクリップボードにコピーします

  • ctrl + shift + c

    コンパイル結果をクリップボードにコピーします

  • alt + w

    エディタの中身を消去します

注意事項

  • 動作のみの場合は必ずif(all)を記述してください

  • 仕様上、余計な文字列などが含まれていても 正しくコンパイルされることがあります

  • ifのネストは最大1層までです。 これを超えると正しくコンパイルされません
